Facebook Rolls Out Dark Mode on Desktop

  • Mohammad Najeeb
  • May 11, 2020
  • Image Source: Tech Crunch



Social media giant Facebook introduced dark mode on the desktop, which is a redesigned version of its desktop website. The new features of facebook.com would be released to all users in the coming weeks.

At last year's F8 developer conference, Facebook CEO Mark Zuckerberg had announced that the company would present a website with a new, optimized design.

You can now use a dark mode on the Facebook desktop so you can use it on your desktop with less brightness, as well as contrast and vibration. If you're using it in low light, screen brightness is minimized.

How to use it?

Users can now switch between Classic mode and Dark mode for desktop. To activate the dark mode, the Facebook desktop user had to go to the Settings option and select Activate Dark Mode from the drop-down option.
